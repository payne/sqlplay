Hello!

https://github.com/payne/sqlplay is a messy experiment in learning spring boot.

This file can be changed without causing a restart....

  1. http://localhost:8080/index.html serves src/main/resources/static/index.html
  2. http://localhost:8080/greeting goes to thymeleaf example with simple form processing!
  3. http://localhost:8080/someOtherUrl goes to the FooServlet. It's a plain HttpServlet.
  4. http://localhost:8080/member?id=42 works because of MemberController
  5. http://localhost:8080/browser/index.html goes to the HAL Browser
  6. http://localhost:8080/h2-console goes to the H2 database console; just click login.
    • Be sure to use the JDBC URL jdbc:h2:mem:testdb
    • This in RAM database is populated by src/main/resources/schema.sql which loads in some CSV files in the same directory.
    • You can use a JDBC URL that points to disk if you like.
